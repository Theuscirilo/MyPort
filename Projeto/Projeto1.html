Sistema de Orçamentos de Gravação – Prefeitura Municipal de Rosário do Ivaí

Este projeto foi desenvolvido com Streamlit e tem como objetivo facilitar o processo de solicitação e gestão de orçamentos para serviços de filmagem realizados ou supervisionados pela Prefeitura Municipal de Rosário do Ivaí. O sistema oferece uma interface amigável tanto para cidadãos solicitarem orçamentos quanto para o acesso administrativo da equipe da Prefeitura.

Formulário de Solicitação

  • Preenchimento de formulário com dados pessoais e detalhes da gravação
  • Armazenamento automático das informações em tempo real

Painel Administrativo

  • Área protegida por senha para funcionários
  • Visualização de todas as solicitações em tabela
  • Exportação dos dados em formato CSV

Persistência Temporária

  • Uso do session_state do Streamlit
  • Sem necessidade de banco de dados externo

Design e Usabilidade

  • Layout limpo e organizado
  • Responsividade adaptada
  • Feedback visual claro

Tecnologias Utilizadas

Python Streamlit Pandas Datetime

Objetivo

Digitalizar o processo de requisição de orçamentos de filmagens, otimizando o fluxo de informações entre os cidadãos e o setor de comunicação da Prefeitura. O sistema oferece mais transparência, praticidade e agilidade na gestão dessas demandas.
