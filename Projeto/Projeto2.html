💊 MyDrugs-Site – Interface Fictícia de E-commerce Inspirada em Série

MyDrugs-Site é um projeto educacional inspirado na série How to Sell Drugs Online (Fast), criado com foco em design, responsividade e estruturação de sites estáticos com HTML, CSS e JavaScript. Trata-se de uma simulação de loja online estilizada, sem qualquer funcionalidade real de vendas, com o objetivo de aprimorar habilidades em desenvolvimento front-end e exibição de projetos web.

🔧 Funcionalidades Principais:

🎨 Interface Moderna e Responsiva:

  • Layout estilizado com cores contrastantes e tipografia destacada.
  • Imagens e banners simulando uma loja online.
  • Design adaptável para dispositivos móveis.

🧭 Navegação Intuitiva:

  • Menu fixo e organizado.
  • Links de navegação para facilitar a experiência do usuário.

💻 Código Limpo e Modular:

  • Separação clara entre HTML, CSS e JavaScript.
  • Estrutura de pastas organizada (assets, imagens, scripts).

🌐 Publicado com GitHub Pages:

  • Projeto acessível publicamente em qualquer dispositivo.
  • Link pronto para compartilhar em currículos e LinkedIn.

🛠 Tecnologias Utilizadas:

  • Linguagem: HTML5, CSS3, JavaScript
  • Hospedagem: GitHub Pages
  • Ferramentas: VS Code, Git, GitHub

💡 Objetivo:

Este projeto foi criado como um exercício de criatividade e domínio técnico, utilizando uma temática fictícia para construir uma loja visualmente atrativa. É uma ótima demonstração de proficiência em front-end, especialmente para iniciantes que desejam ter um projeto completo no GitHub com deploy ativo e boa apresentação visual.
