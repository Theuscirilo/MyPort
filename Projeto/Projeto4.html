Sistema de Atendimento Urbano

Sistema de Atendimento Urbano – Prefeitura Municipal de Rosário do Ivaí é um projeto desenvolvido com Streamlit com o objetivo de facilitar a comunicação entre os cidadãos e a Prefeitura Municipal de Rosário do Ivaí. A plataforma permite que moradores relatem problemas urbanos, façam solicitações de serviço ou enviem sugestões diretamente à equipe administrativa. O sistema é simples, responsivo e totalmente digital, promovendo mais agilidade e transparência no atendimento público.

Formulário de Solicitação Cidadã

  • Envio de pedidos com nome, contato, localidade, endereço e descrição do problema.
  • Opções para tipos de problema: buracos, iluminação, entulho, árvores e outros.
  • Armazenamento em tempo real usando session_state.

Área Administrativa

  • Acesso restrito por senha para a equipe da Prefeitura.
  • Visualização de pedidos em tabela interativa.
  • Exportação para arquivo CSV.
  • Logout seguro da sessão administrativa.

Persistência Temporária

  • Gerenciamento dos dados do usuário enquanto a aplicação estiver ativa.
  • Não utiliza banco de dados externo — ideal para protótipos.

Interface Intuitiva

  • Layout amigável com separação entre área pública e área administrativa.
  • Totalmente responsivo, com mensagens claras e intuitivas.

Tecnologias Utilizadas

Python Streamlit Pandas

Objetivo

Digitalizar o canal de atendimento urbano da Prefeitura, oferecendo uma solução prática para que os moradores comuniquem problemas e colaborem com a gestão pública. O sistema reduz burocracias, agiliza a triagem de demandas e fortalece o vínculo entre administração e população.
